# Game Rules 
1. Use the arrow buttons to hop from square to square 
2. Try to avoid the bugs. 
3. If you hit a bug your position will be reset.  
4. The game is over when you reach the water.  

# Make with 
1. HTML, CSS, and JavaScript
2. HTML 5 Canvas
